Glossário Dev por Pedro Leite Fiuza

Um guia de referência para tags HTML, propriedades CSS e conceitos de JavaScript.

HTML

<div>

A tag <div> (divisão) é um contêiner genérico para agrupar outros elementos HTML. É frequentemente usada para aplicar estilos com CSS ou para manipular com JavaScript.

<p>

A tag <p> define um parágrafo. Os navegadores adicionam automaticamente um espaço (margem) antes e depois de cada elemento <p>.

<a>

A tag <a> (âncora) cria um hyperlink para outras páginas da web, arquivos, locais na mesma página ou endereços de e-mail.

<img>

A tag <img> é usada para incorporar uma imagem em uma página HTML. Requer o atributo `src` para especificar o caminho da imagem.

<span>

Similar à <div>, mas é um contêiner em linha (inline). Usado para agrupar elementos em linha para estilização ou manipulação, sem quebrar o fluxo do texto.

<ul> e <li>

A tag <ul> (Unordered List) cria uma lista não ordenada (com marcadores). Cada item da lista é definido pela tag <li> (List Item).

<ol>

A tag <ol> (Ordered List) cria uma lista ordenada (numerada). Assim como na lista não ordenada, os itens são definidos pela tag <li>.

<h1> a <h6>

As tags de cabeçalho definem títulos e subtítulos, onde <h1> é o mais importante e <h6> é o menos importante. São fundamentais para a semântica e SEO.

<form>

Define um formulário HTML para a entrada de dados do usuário. Pode conter elementos como <input>, <textarea>, <button>, etc.

<table>

Cria uma tabela para exibir dados em formato de linhas e colunas. Utiliza tags como <tr> (linha), <th> (cabeçalho) e <td> (célula de dados).

CSS (Cascading Style Sheets)

color

A propriedade color é usada para definir a cor do texto de um elemento.

background-color

A propriedade background-color define a cor de fundo de um elemento.

font-size

A propriedade font-size define o tamanho do texto.

display

A propriedade display especifica o tipo de caixa de renderização usada por um elemento. Valores comuns incluem `block`, `inline`, e `flex`.

margin

A propriedade margin é usada para criar espaço ao redor dos elementos, fora de qualquer borda definida. É o espaçamento externo.

padding

A propriedade padding é usada para criar espaço ao redor do conteúdo de um elemento, dentro de qualquer borda definida. É o espaçamento interno.

border

A propriedade border é uma forma abreviada para definir a largura, o estilo e a cor da borda de um elemento em uma única declaração.

position

Define como um elemento é posicionado no documento. Valores como `static`, `relative`, `absolute`, `fixed` e `sticky` alteram o fluxo do elemento.

flexbox

Um módulo de layout que oferece uma maneira mais eficiente de alinhar e distribuir espaço entre itens em um contêiner, mesmo quando seus tamanhos são desconhecidos.

grid

O CSS Grid Layout é um sistema de layout bidimensional que permite organizar o conteúdo em linhas e colunas, ideal para layouts complexos.

JavaScript

let

A palavra-chave let declara uma variável com escopo de bloco, opcionalmente, inicializando-a com um valor.

const

A palavra-chave const declara uma variável de escopo de bloco, somente leitura. Uma vez atribuído um valor, ele não pode ser reatribuído.

function

A palavra-chave function é usada para declarar uma função, que é um bloco de código projetado para executar uma tarefa específica.

addEventListener

O método addEventListener anexa um manipulador de eventos a um elemento, permitindo que você execute uma função quando um evento específico ocorre (como um clique).

if / else

Estruturas de controle condicional. O if executa um bloco de código se uma condição for verdadeira; o else executa outro bloco se a condição for falsa.

for

Cria um laço (loop) que consiste em três expressões opcionais, entre parênteses e separadas por ponto e vírgula, seguidas por uma declaração a ser executada no laço.

Array

Um objeto global usado na construção de 'arrays': objetos de alto nível, semelhantes a listas, que armazenam múltiplos valores em uma única variável.

Object

Uma coleção de propriedades, onde cada propriedade é uma associação entre uma chave (ou nome) e um valor. É uma das estruturas de dados fundamentais em JS.

DOM

O Document Object Model (DOM) é uma interface de programação para documentos HTML. Ele representa a página para que os programas possam alterar a estrutura, estilo e conteúdo do documento.

querySelector

O método querySelector() retorna o primeiro elemento dentro do documento que corresponde ao seletor ou grupo de seletores especificado.
